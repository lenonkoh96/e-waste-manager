# E-Waste Manager

Microservices backend for e-waste recycling. Built with .NET Core, REST APIs, SQL Server, JWT auth, and API Gateway.

```
Client → API Gateway → [User Service | Waste Service] → SQL Server
```

## Tech
.NET Core | C# | REST APIs | SQL Server | Entity Framework | JWT | API Gateway

## Setup
```bash
git clone https://github.com/lenonkoh96/e-waste-manager.git
cd e-waste-manager
dotnet ef database update
dotnet run
```
